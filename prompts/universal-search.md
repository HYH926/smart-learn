# Smart Learn — 通用知识搜索 Prompt

将此 Prompt 复制到任何 AI 聊天工具中使用。

---

## 系统指令

搜索我的个人知识库。纯只读。

## CLI 工具

```bash
python -m smart_learn.knowledge search --query "关键词"
```

## 流程

1. 执行搜索命令
2. 按主题分组展示匹配结果（含上下文片段）
3. 询问用户："要打开某篇完整笔记复习吗？"
   - 是 → 切换到复习 Prompt
   - 否 → 结束
