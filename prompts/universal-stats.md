# Smart Learn — 学习统计 Prompt（完整版）

将此 Prompt 复制到任何 AI 聊天工具中使用。

---

## 系统指令

读取我的知识库，展示学习统计面板。纯只读。

## CLI 工具

```bash
python -m smart_learn.knowledge stats
```

## 统计维度

### 1. 学习概览
- 📚 总学习主题数
- 🧠 总概念数
- ⚠️ 总薄弱点数
- ✅ 最近 7 天已复习数
- 🕐 超过 14 天未复习数
- 📈 复习率

### 2. 主题详情表
| 主题 | 学习日期 | 概念数 | 薄弱点 | 核心公式 |
|------|---------|--------|--------|---------|

### 3. 薄弱点总览（按最久未复习排序）
```
⚠️ 薄弱点分布
距今天数  主题          薄弱点
────────────────────────────────
  N天     主题A         薄弱点1
  N天     主题B         薄弱点2
```
提示："🕐 超过 14 天的建议用复习 Prompt 回顾"

### 4. 知识库路径
- 📝 笔记目录：knowledge_store/
- 🧠 思维导图：knowledge_store/*_思维导图.md
- 📄 Word 文档：knowledge_store/*_学习笔记.docx（如有）
